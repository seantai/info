はじめに

BIM開発を一緒に学ぶ#

🌱 プログラミングを全く知らないが、これから始めたいという方も、経験豊富な開発者だが、BIMアプリケーションを次のレベルに引き上げたいという方も、ぜひご利用ください。

🚀 すべてのレベル向けのBIMコースを提供しています。これらのコースは以下の通りです。

📺 英語での徹底的なビデオレッスン。
👩‍💻 全説のスクリプト。
📋 全コードにコメント付き。
👩‍🏫 先生方からの回答。
🏆 限定ハッカソンへのアクセス。
🎁 IFC.jsの独占マーチャンダイジング。

なぜ、これらのコースは無料ではないのですか？#

🤝 IFC.jsは現在も、そしてこれからも、誰でも無料で利用できます。 しかし、プロジェクトを成長させ続けるためには、イベントの開催、プロジェクト費用の支払い、活発なコミュニティへの報酬など、多くの資金が必要です。これらのコースは、そのような資金を得るための方法です。

🌈 講座で得た資金は全てプロジェクトに再投資されます。完全に透明な方法で資金を管理している様子は、Open Collectiveでご覧いただけます。

👨‍💻 このお金をどこかに使ってほしいですか？ スポンサー または パトロン としてプロジェクトに参加し、私たちがこの資金を使う場所について発言権を持つことができます。

価格はいくらですか？#

非パトロールの場合#

🧑 各コースとも、パトロン以外の方は100ドルです。しかし、そのお金は私たちに行くのではなく、プロジェクトに行くのです。したがって、コースを購入するには、オープンコレクティブで100ドルの寄付をするとバウチャーとアクセスしたいコース（複数可）を送るだけでよいのです。

パトロンの皆様へ#

🧑🔥 パトロンは、パトロン会員数に応じて、お得な割引価格でコースを購入することができます。パトロンを1ヶ月間継続するごとに、すべてのコースが15ドル割引され、最大25ドルまで購入できます。

🎉 つまり、6ヶ月間パトロンに留まることを約束したコミュニティメンバーは、どのコースでも25ドルで購入することができます。

🏆 さらに、一般ユーザーがアクセスできない独占コンテンツや、次回の講座のテーマを決定する権限も持つことになります。

💛 もちろん、パトロンにはパトロンの階層に含まれる他のすべてのメリットもあります。

企業パトロンの皆様へ#

👨‍💼💼 法人パトロンには、社員50名までの全コースへの無料アクセスのほか、ライブラリのクリエイターから開発チームへのライブウェビナー、プライムサポートなど、法人パトロンが付与するすべての特典を提供します。

アダプティブディスカウント#

🤝 世界は多様であり、すべての国が同じ購買力を持っているわけではないことは承知しています。しかし、世界的な経済的理由で人々を省みたくはありません。もしあなたの国の平均給与が1000ドル以下であれば、適応割引を適用することができます。

😊 この割引は、あなたの国の平均給与と$1000との**比例した差に相当する価格で、コースとパトロン階層にアクセスすることができます。例えば、平均給与が800ドルであれば、20％の割引となります。平均給与が400ドルであれば、60％の割引となります。

どうすれば、良いものだとわかるのでしょうか？#

🚀 ご安心ください。最初のレッスンとノートは、無料でどなたでもご利用いただけますので、お金を払う前に他の教材の品質を試すことができます。

入会するにはどうしたらいいですか？#

🚧 コースはまだできていません 一刻も早く実現できるよう、一生懸命取り組んでいます。その間、パトロンになると、それらが出たときに素晴らしい割引を受けることができます（上記の通り）。私たちは、それらが3月/4月にリリースされることを見積もっています。

👌 リリースされたら、私たちのopen-collective accountに寄付していただければ、全てにアクセスできるようになります。.
