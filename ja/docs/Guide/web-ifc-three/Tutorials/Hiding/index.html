👓 隠蔽

はじめに#

👩‍🏫 他のチュートリアルで見てきたように、3Dで要素を選択し、そのプロパティにアクセスし、モデル全体の一部であるサブセットを使用してハイライトする方法は既に知っています。

🚪🧱 しかし、多くのBIMアプリケーションでは、要素を隠したり分離したりすることも可能です。一般的な使用例としては、特定のフィルターに準拠していないすべての要素を非表示にしたり、建物のある階に属するオブジェクトのみを表示したりすることが挙げられます。

🍰 もちろん、IFC.jsを使えば、こんなことは朝飯前です。この例では、ユーザーがチェックボックスを使ってアイテムを表示したり隠したりできるように、カテゴリー別のフィルターを作成することにします。

他のチュートリアルと同様に、完全なサンプルはこちらでご覧になれます。

方法#

カテゴリの部分集合を取得する#

👩‍🏫 IFC.jsの可視性は、サブセット操作をベースにしています。これにより、複雑なビジュアライゼーションを最小限のメモリ使用量で作成することができます。最初に行うことは、ユーザーが表示または非表示を許可する IFC カテゴリ を定義することです。

📋📋📋 メモリ節約のため、IFC.jsのカテゴリーは数値定数として定義されています。そこで、これらの定数の名前と数値を対応させるオブジェクトと、それを取得する関数を作ってみましょう。

import {
IFCWALLSTANDARDCASE,
IFCSLAB,
IFCDOOR,
IFCWINDOW,
IFCFURNISHINGELEMENT,
IFCMEMBER,
IFCPLATE
} from 'web-ifc';
// カテゴリ名一覧
const categories = {
IFCWALLSTANDARDCASE,
IFCSLAB,
IFCFURNISHINGELEMENT,
IFCDOOR,
IFCWINDOW,
IFCPLATE,
IFCMEMBER
};
// カテゴリ名を取得する
function getName(category) {
const names = Object.keys(categories);
return names.find(name => categories[name] === category);
}

🏠⏩🚪 では、あるカテゴリに属する要素のすべてのIDを取得し、それらのIDを持つサブセットを作成する関数をいくつか作ってみましょう。

💡
IFC.jsの関数の多くは非同期であることを思い出して、asyncとawaitを使うことにします。

👩‍🏫 また、removeFromSubset() を使用すると、サブセットからひとつの項目を削除することができます (たとえば、ひとつの項目を非表示にすることができます)。これを createSubset() (removePrevious = false) と組み合わせると、どのサブセットに何を追加するか、またその可視性を完全に制御することができます。

// 指定されたカテゴリの全項目のIDを取得する
async function getAll(category) {
const manager = ifcLoader.ifcManager;
return manager.getAllItemsOfType(0, category, false);
}
// カテゴリのすべての要素を含む新しいサブセットを作成します。
async function newSubsetOfType(category) {
const ids = await getAll(category);
return ifcLoader.ifcManager.createSubset({
modelID: 0,
scene,
ids,
removePrevious: true,
customID: category.toString()
})
}

GUIを視認性でバインドする#

✨ あとは、各チェックボックスのクリックイベントと対応するカテゴリの可視性をリンクさせ、チェックボックスがアクティブになるとカテゴリが可視化され、非アクティブになるとカテゴリが消滅するようにするだけです。

🎨👨‍🎨 ここで注目すべきは、コードをより簡潔にするために、各チェックボックスに、リンク先のカテゴリと同じ名前のIDを与えていることです。

// 作成したサブセットを格納する
const subsets = {};
async function setupAllCategories() {
const allCategories = Object.values(categories);
for (let i = 0; i < allCategories.length; i++) {
const category = allCategories[i];
await setupCategory(category);
}
}
// 新しいサブセットを作成し、チェックボックスを設定する
async function setupCategory(category) {
subsets[category] = await newSubsetOfType(category);
setupCheckBox(category);
}
// チェックボックスイベントによる要素の表示／非表示を設定する
function setupCheckBox(category) {
const name = getName(category);
const checkBox = document.getElementById(name);
checkBox.addEventListener('change', (event) => {
const checked = event.target.checked;
const subset = subsets[category];
if (checked) scene.add(subset);
else subset.removeFromParent();
});
}
