パトロン

コミュニティーに参加する#

🌱 あなたは、私たちの旅に参加し、私たちのBIM＆開発ラーニングコミュニティの積極的な一部になりたいですか？それなら、これはあなたが探していたものです。

代わりに開発チームに参加したい場合は、コントリビューションページにアクセスしてください。.

個人として#

🧑 パトロンのシステムはとてもシンプルで、IFC.jsが成長し続けるために5ドル/月をプロジェクトに寄付していただきます。しかし、私たちは、オープンソースが利他的な寄付によって左右されるべきではないと信じています。このため、すべてのパトロンには、以下のものを提供します。

👩‍🏫

BIMプログラミングコース（全レベル対象

あらゆるレベルに対応したBIMプログラミングコースがあります。各コースは100ドルですが、パトロンになるとわずか25ドルでアクセスできます。

👩‍👩‍👦‍

パトロンコミュニティへのアクセス

パトロンは、私たちに直接質問や提案をしたり、他のアクティブなメンバーや企業と出会うことができる専用チャンネルにアクセスすることができます。

🎥

独占ライブウェビナーおよびストリーム

毎週、チュートリアル、ライブ・コーディング・セッション、インタビュー、パトロン・コミュニティ・プロジェクトの紹介など、パトロン限定のストリームを配信しています。

🤝

お客様のプロジェクトをサポートします

パトロンの皆様には、プロジェクトに関するご相談をお受けしています。私たちは日々BIMソフトウェアを開発しており、その経験を共有することで、パトロンがBIMソフトウェアを次のレベルへ引き上げることができます。

🏆

ハッカソンへの優先参加権

パトロンだけが参加できる豪華賞品（現金や商品）が当たるハッカソンを開催します。このハッカソンは、私たちのネットワークで放送され、インタビューも行われます。

🚀

ネットワークからの視認性

パトロンおよびパトロンが開発するBIM製品は、私たちのネットワークを利用して製品を展示し、ネットワークを拡大することができます。

🎁

商品・特典

パトロンには、IFC.js限定のマーチャンダイズやギフトが贈られます。

企業として#

👨‍💼 私たちは、月々50ドルのコーポレート・パトリオンを持っています。上記の特典に加え、以下の特典があります。

👨‍💼

コンサルティング、プライムサポート

プロフェッショナルなBIMツール制作の経験を貴社と共有し、最適なソリューションのご提案をさせていただきます。

👩‍🏫

全コース無料＋プライベートウェビナー

IFC.jsの全公式コースが無料で受講できるほか、専用のWebセミナーやコンサルティングを受けることができます。

🚀

ネットワークを活用した製品プロモーション

あなたのBIMアプリケーションは、ハッカソンやストリームインタビューなど、私たちのすべてのネットワークで宣伝されます。

💪

意思決定力

書店の次のステップをあなたのビジネスと一致させるために、ある程度の決定権を持つことになります。

私のお金はどこに行くのですか？#

💛 私たちはすべての資金をOpen Collectiveで管理しています。つまり、誰がどれだけのお金をプロジェクトに投入したかを誰でも見ることができるのです。

💵 そのお金を使いたいときは毎回、同じページで公開される請求書をアップロードしなければなりません。そうすれば、お金はコードと同じようにオープンです。
