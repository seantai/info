开始工作

与我们一起学习BIM开发#

🌱 无论你对编程一无所知，但你想开始学习，还是你是一个有经验的开发者，但想把你的BIM应用提高到一个新的水平，我们都能满足你的要求。

🚀 我们提供所有级别的BIM课程。这些课程包括。

📺 深入的英语视频课程。
👩‍💻 所有理论的脚本。
📋 所有的注释代码。
👩‍🏫 教师的回答。
🏆 参加独家黑客活动。
🎁 独家IFC.js商品销售。

为什么这些课程不是免费的？#

🤝 IFC.js对每个人都是免费的，而且永远都是。但是，为了继续发展这个项目，我们需要资金来组织活动，支付项目费用，奖励活跃的社区等等。这些课程是获得这些资金的一种方式。

🌈 来自课程的资金完全再投资于该项目。你可以在开放集体看到我们如何以完全透明的方式管理资金。

👨‍💻 你希望我们把这些钱花在别的地方吗？你可以作为赞助商或赞助人加入项目，对我们把这些资金用在哪里有发言权。

它们的成本是多少？#

对于非剧院#

🧑 每门课程的费用为$100，非剧院。但这些钱并不归我们所有，而是归项目所有。因此，要购买一门课程，你只需要公开集体捐款100美元，并把凭证以及你想获得的课程（或课程）寄给我们。

对于赞助人#

🧑🔥 赞助人可以根据他们的赞助人资格，以很大的折扣购买课程。每不间断地在patreon中停留一个月，所有的课程都会打折15美元，最高可达25美元。

🎉 换句话说，承诺在patreon呆6个月的社区成员可以用25美元购买任何课程。

🏆 此外，他们将拥有普通用户无法接触到的独家内容，并有权决定下一个课程的主题。

💛 当然，赞助人也有包含在赞助人级别中的所有其他优势。

对于企业赞助者#

👨‍💼💼 企业赞助人有免费使用所有课程的权利，最多可容纳50名员工，以及来自图书馆创建者对其开发团队的现场网络研讨会，主要支持和所有其他由企业赞助人授予的好处。

适应性折扣#

🤝 我们知道，世界是多元化的，并非所有国家都有相同的购买力。然而，我们不希望因为全球经济原因而遗漏了一些人。如果你所在国家的平均工资低于1000美元，你可以适用适应性折扣。

😊 这种折扣使人们能够以相当于你所在国家的平均工资与1000美元之间的比例的价格获得课程和赞助层级。例如，如果平均工资是800美元，你可以得到20%的折扣。如果平均工资是400美元，折扣将是60%。

我怎么知道它们是好的呢？#

🚀 不用担心! 第一批课程和笔记是免费的，并对任何人开放，所以你可以在支付任何费用之前尝试其余材料的质量。

我如何注册？#

🚧 这些课程还没有准备好! 我们正在非常努力地工作，以使它们尽快实现。在此期间，你可以成为赞助人，并在它们出来时得到一个很大的折扣（如上所述）。我们估计它们将在**三月/四月发布。

👌 一旦它们被释放，请向我们的开放式集体账户捐款，我们将让你获得一切。
